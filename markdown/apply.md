# apply





* * *

### apply.apply(fn, args) 

Apply a function to an argument list.

**Parameters**

**fn**: `function`, Apply a function to an argument list.

**args**: `array`, Apply a function to an argument list.

**Returns**: `any`

**Example**:
```js
let apply = require('102f/src/apply')
let arr = [0, 1, 2, 3]
apply(Math.max, arr) // 3
```
