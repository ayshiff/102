# multiply





* * *

### multiply.multiply(a, b) 

Returns the multiplication of two Numbers.

**Parameters**

**a**: `Number`, Returns the multiplication of two Numbers.

**b**: `Number`, Returns the multiplication of two Numbers.

**Returns**: `Number`

**Example**:
```js
var a = 2
var b = 3
var result = multiply(a)(b) //6
```
